# e-plantShopping

Paradise Nursery is a React shopping cart application for an online house plant shop. The app includes a landing page, a categorized product listing page, and a shopping cart page with quantity controls powered by Redux Toolkit.

## Features

- Landing page with company information and a Get Started button
- Navbar links for Home, Plants, and Cart
- Three plant sections: Aromatic Plants, Medicinal Plants, and Air Purifying Plants
- Add to Cart buttons that disable after a plant is added
- Dynamic cart icon that updates with the total number of items
- Cart page with increase, decrease, delete, continue shopping, and checkout actions

## Tech Stack

- React
- React Router
- Redux Toolkit
- React Redux
- Vite

## Scripts

```bash
npm install
npm run dev
npm run build
```
